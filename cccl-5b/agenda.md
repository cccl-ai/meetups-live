## About

For software engineers using Claude Code in production — beyond basic code generation. Six 10-minute talks and demos on AI-powered workflows across the SDLC.

Organised by Vikram Pawar (Claude Community Ambassador), co-hosted with Rob Hart (AI for Engineers London). Speakers from Figma, Google, CrowdStrike, and the community.
